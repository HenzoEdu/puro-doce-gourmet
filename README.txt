PURO DOCE GOURMET - V8.2

COMO TESTAR NO PC
1. Extraia o .zip
2. Abra o arquivo index.html no navegador
3. O app já funciona offline com armazenamento local no navegador

PARA TESTAR COMO PWA NO IPHONE
1. Hospede a pasta em um serviço estático (GitHub Pages, Netlify, Vercel ou Cloudflare Pages)
2. Abra o link no Safari do iPhone
3. Toque em Compartilhar > Adicionar à Tela de Início
4. Se já tinha instalado versão antiga, apague o ícone antigo e adicione novamente

RECURSOS INCLUSOS
- 100% sem backend
- Dados persistem ao fechar e abrir
- Estoque, vendas e financeiro separados
- Exportar e importar backup JSON
- Reset geral
- Filtro de históricos
- Visual com logo e ícones da marca

OBSERVAÇÃO
Para evitar bugs de tela preta e cache quebrado no iPhone, esta versão não usa service worker.
